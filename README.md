## Environment variables

Optional env vars are listed in `env.example` (copy to `.env` for local use). For the contact form to actually send messages, set:

- **`CONTACT_API_URL`** (server-only): Full URL of your backend contact/submit endpoint. The app will POST JSON `{ name, email, subject, message }` to this URL after validating the math challenge. If unset or empty, the contact form returns a 503 with a clear message asking users to use the provided email instead.
